## 📌 Descrição
Este projeto foi desenvolvido em grupo com o objetivo de simular um ambiente real de desenvolvimento utilizando GitHub.

O site criado é um **Portal de Jogos**, contendo informações sobre jogos, categorias e um botão fictício de interação.
